WhatsApp Cloud API Setup

Purpose
- Language-aware routing to WhatsApp groups
- Optional DFY outreach operations

Setup
1) Create Meta developer app
2) Add WhatsApp product
3) Get:
   - Phone number ID
   - WhatsApp Business Account ID
   - Permanent access token
4) Configure:
   - WHATSAPP_TOKEN
   - WHATSAPP_PHONE_NUMBER_ID

Platform behavior
- Auto-detect country + language
- Suggest best group from group catalog
- Provide approved pitch templates
